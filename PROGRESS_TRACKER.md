# 学习进度追踪表

> 建议将本表复制到自己的笔记工具（Notion、飞书文档、OneNote 等），或打印后手写勾选。完成每一课的同时，记录练习和复盘情况，便于后续查漏补缺。

## 使用方式

1. 勾选“✅”栏表示本课整体任务（阅读 + 示例 + 练习）已完成
2. “练习完成”用于确认所有 `exercises.py` 中的题目是否全部实现
3. “复盘/笔记”填写自己的总结位置（文件名、笔记链接或手写页码）
4. “完成日期”记录实际完成时间，帮助评估节奏是否合适

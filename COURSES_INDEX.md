## 📂 课程文件结构

每个课程目录包含以下文件：

### Python基础课程
```
lessonXX-topic-name/
├── README.md          # 课程说明文档
├── demo.py            # 示例代码
└── exercises.py       # 练习题
```

### PyTorch课程
```
lessonXX-topic-name/
├── README.md          # 课程说明文档
├── pytorch_demo.py    # PyTorch示例代码
└── exercises.py       # 练习题
```

## 📝 学习建议

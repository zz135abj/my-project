# PyTorch深度学习框架教程

本部分基于陈云著《PyTorch:入门与实践》，带你深入学习PyTorch深度学习框架，预计学习时间3-6周。

## 📋 课程大纲

### 第3-4周：PyTorch基础和张量操作

#### 第15课：PyTorch简介与环境搭建
- PyTorch是什么？
- PyTorch vs TensorFlow
- 环境搭建与安装
- 第一个PyTorch程序
- **练习**：安装PyTorch并运行基础示例

#### 第16课：张量基础
- 张量的概念
- 张量的创建方法
- 张量的属性
- **练习**：创建不同类型和大小的张量

#### 第17课：张量操作
- 张量的索引和切片
- 张量的形状变换
- 张量的数学运算
- **练习**：张量运算练习

#### 第18课：张量与NumPy转换
- 张量与NumPy数组的相互转换
- GPU张量操作
- **练习**：GPU加速计算实验

#### 第19课：自动求导机制
- autograd原理
- 梯度计算
- 计算图概念
- **练习**：手动计算梯度对比

#### 第20课：数据处理工具
- Dataset和DataLoader
- 数据预处理
- 数据增强
- **练习**：自定义数据集类

#### 第21课：神经网络基础
- nn.Module介绍
- 常用层类型
- 模型参数管理
- **练习**：构建简单神经网络

### 第5-6周：神经网络基础

#### 第22课：损失函数
- 常用损失函数
- 损失函数的选择
- **练习**：不同损失函数对比

#### 第23课：优化器
- 梯度下降算法
- 常用优化器
- 学习率调整
- **练习**：优化器性能比较

#### 第24课：训练循环
- 完整训练流程
- 模型评估
- 早停机制
- **练习**：实现完整的训练循环

#### 第25课：模型保存与加载
- 模型保存方法
- 模型加载与恢复
- 迁移学习基础
- **练习**：模型保存加载实验

#### 第26课：卷积神经网络(CNN)
- CNN原理
- 卷积层和池化层
- 经典CNN架构
- **练习**：构建CNN分类器

#### 第27课：循环神经网络(RNN)
- RNN原理
- LSTM和GRU
- 序列数据处理
- **练习**：文本分类任务

#### 第28课：实战项目：图像分类
- 数据集准备
- 模型设计
- 训练与调优
- **练习**：完成完整的图像分类项目

## 🎯 学习目标

完成本部分学习后，你将：
- 掌握PyTorch框架的核心概念

- 具备解决实际AI问题的能力

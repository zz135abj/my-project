# Python与PyTorch入门教程

专为编程新手设计的Python基础和PyTorch深度学习框架学习项目。

## ✨ 课程特色

- 📚 **完整的课程体系**：28课完整课程，从零基础到深度学习
- 💻 **丰富的代码示例**：每课都有详细的示例代码和注释
- ✏️ **大量练习题**：100+练习题帮助巩固知识
- 🎯 **循序渐进**：课程难度逐步递增，适合自学

## 🧭 新手快速导航

1. 阅读 [BEGINNER_GUIDE.md](./BEGINNER_GUIDE.md) 了解推荐的学习节奏与技巧
2. 按照 [SETUP.md](./SETUP.md) 完成环境配置（建议使用虚拟环境）
3. 使用 [PROGRESS_TRACKER.md](./PROGRESS_TRACKER.md) 记录每课完成情况与复盘
4. 准备好之后，从 [python-basics](./python-basics/) 的 lesson01 开始循序渐进学习

## 🔄 推荐学习流程

1. **预习**：先阅览 [COURSE_OUTLINE.md](./COURSE_OUTLINE.md) 和本课 README，明确目标
2. **学习**：按照课程说明运行示例代码，理解每个步骤的作用
3. **实践**：独立完成 `exercises.py` 中的练习，尝试扩展或变体
4. **复盘**：在笔记或进度表中总结收获和疑问，隔天再回顾一次巩固记忆

> 建议结合番茄工作法（25 分钟专注 + 5 分钟整理）保持稳定节奏。

## 📚 学习大纲

### 第一部分：Python基础编程 (第1-14课)
- **课程数量**：14课完整课程
- **学习时长**：约42小时
- **目标**：掌握Python基础语法和面向对象编程
- **适合人群**：零基础编程学习者
- **学习方式**：理论学习 + 实践练习

- **课程数量**：14课完整课程

## 🚀 开始学习

1. 克隆本项目到本地
2. 查看[课程索引](./COURSES_INDEX.md)了解所有28课的详细信息
3. 按照[Python基础](./python-basics/)部分开始学习（第1-14课）
4. 完成Python基础后，进入[PyTorch教程](./pytorch-tutorial/)部分（第15-28课）
5. 每节课都要完成对应的练习题

### 快速开始示例
```bash
# 第一课：Python简介
cd python-basics/lesson01-hello-world
cat README.md

# 第十五课：PyTorch简介
cd pytorch-tutorial/lesson15-introduction-setup

```

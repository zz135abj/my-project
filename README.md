# Python与PyTorch入门教程

专为编程新手设计的Python基础和PyTorch深度学习框架学习项目。

## 📚 学习大纲

### 第一部分：Python基础编程 (1-2周)
- **目标**：掌握Python基础语法和面向对象编程
- **适合人群**：零基础编程学习者
- **学习方式**：理论学习 + 实践练习

### 第二部分：PyTorch深度学习框架 (3-6周)
- **目标**：掌握PyTorch框架，能够构建简单的神经网络
- **参考教材**：《PyTorch:入门与实践》(陈云著)
- **学习方式**：框架学习 + 项目实战

## 🎯 学习目标

完成本教程后，你将能够：
1. 熟练使用Python进行编程
2. 理解面向对象编程概念
3. 掌握PyTorch深度学习框架
4. 能够构建和训练简单的神经网络模型
5. 具备继续深入学习AI的基础

## 📅 学习计划

| 周数 | 学习内容 | 预计时间 |
|------|----------|----------|
| 第1周 | Python基础语法 | 7-10小时 |
| 第2周 | Python面向对象编程 | 7-10小时 |
| 第3-4周 | PyTorch基础和张量操作 | 10-15小时 |
| 第5-6周 | 神经网络基础 | 10-15小时 |
| 第7-8周 | 实战项目 | 15-20小时 |

## 🚀 开始学习

1. 克隆本项目到本地
2. 按照[Python基础](./python-basics/)部分开始学习
3. 完成Python基础后，进入[PyTorch教程](./pytorch-tutorial/)部分
4. 每节课都要完成对应的练习题

## 📋 学习建议

- 每天保持1-2小时的学习时间
- 理论学习后立即动手实践
- 遇到问题时先尝试自己解决，再查看答案
- 建立学习笔记，记录重要概念和代码

## 🛠️ 环境要求

- Python 3.8+
- PyTorch 1.0+
- Jupyter Notebook (推荐)
- 基本的文本编辑器
